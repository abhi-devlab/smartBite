
# SmartBite – AI-Powered Personalized Meal Planner

SmartBite is an AI-driven personalized meal planning platform developed as our B.Tech Final Year Project. The system helps users generate customized 7-day meal plans based on their age, height, weight, health conditions, allergies, dietary preferences, activity level, and budget. Unlike many existing nutrition applications, SmartBite focuses on Indian cuisine, seasonal fruits, and affordable food choices to provide practical and culturally relevant diet recommendations.

The platform combines Machine Learning and Generative AI to deliver intelligent nutrition guidance. An XGBoost model analyzes user data and food datasets to recommend suitable meals, while an AI-powered health chatbot assists users with diet and wellness-related queries. The application also includes secure JWT-based authentication, responsive React frontend interfaces, and a FastAPI-powered backend for seamless user experience.

### Key Features

* AI-generated personalized 7-day meal plans
* Indian cuisine-focused recommendations
* Seasonal fruit and local food suggestions
* Budget-friendly nutrition planning
* XGBoost-based recommendation engine
* AI health and diet chatbot
* JWT authentication and secure login system
* Responsive React frontend
* FastAPI backend integration
* Personalized calorie and nutrition tracking

### Tech Stack

### Project Goal

The primary objective of SmartBite is to make healthy eating more accessible by providing personalized, affordable, and culturally relevant meal recommendations. The platform bridges the gap between modern AI-powered nutrition systems and the dietary needs of Indian users.

This project was developed as part of the Bachelor of Technology (B.Tech) curriculum and demonstrates the practical application of Machine Learning, Artificial Intelligence, Full Stack Development, and Healthcare Technology in solving real-world nutrition challenges.
